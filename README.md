# Vanilla JS boilerplate with Parcel, Babel and Sass

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Clone

```bash
git clone https://github.com/weyvern/js-boilerplate.git
```

## Install

```bash
npm install
```

## Dev enviroment

```bash
npm run dev
```

## Build

```bash
npm run build
```
